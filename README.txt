# #🎣 Информационно-навигационная система «Fishing Map Pro»

**Автор проекта:** schuller99

## 📄 Описание проекта
**Fishing Map Pro** — это интерактивная веб-ориентированная геоинформационная система (ГИС), созданная для автоматизации и упрощения процесса планирования любительской рыбной ловли. Проект решает проблему фрагментированности данных, объединяя картографию, навигацию, метеорологию и законодательную базу в едином интерфейсе, оптимизированном для работы в полевых условиях.

## 🎯 Цели и задачи
**Цель проекта:** Создать надежный, автономный (вычисления на стороне клиента) и удобный веб-инструмент для рыболовов Саратовской области, который повысит результативность лова, безопасность на воде и правовую осведомленность граждан.

**Ключевые задачи:**
* Визуализировать геоданные (базовые точки лова) с цветовой градацией по видам рыб.
* Интегрировать картографическую платформу со сторонними навигационными системами (Яндекс.Карты, 2ГИС).
* Реализовать автоматический запрос погодных условий (температура, ветер) через сторонние API в реальном времени.
* Внедрить систему оповещений об актуальной правовой базе (правила Рыбнадзора, запретные периоды, нормы вылова).
* Разработать локальную базу данных для сохранения пользовательских меток (ям) без использования серверного бэкенда.

## 📊 Анализ центральной аудитории (ЦА)
Разработка логики работы и пользовательского интерфейса базируется на результатах парсинга и анализа базы данных рыболовного сообщества Саратовского региона. Согласно обработанной статистике:

* **Общая база (потенциальные пользователи):** 32 532 человека.
* **Активная аудитория ("живые" аккаунты):** 25 245 пользователей.
* **Гендерное распределение:** 87.0% мужчин и 13.0% женщин.
* **Средний возраст ЦА:** 41.3 года.

*Практический вывод:* Учитывая высокий средний возраст (41+ год) и специфику применения (использование со смартфона, нахождение в лодке, яркое солнце, мокрые руки), UI/UX дизайн спроектирован по принципу максимальной утилитарности. Элементы управления сделаны крупными, шрифты — контрастными, а все критически важные функции (построение маршрута, информация о законах) доступны в 1–2 клика.

## 🌟 Функциональные модули
1. **Модуль маршрутизации:** Построение маршрутов до точек лова с использованием внутренних скриптов (OSRM API) и генерации deep-ссылок для автоматического запуска навигаторов на смартфоне.
2. **Правовой модуль (Рыбнадзор):** Интеллектуальный статус-бар, учитывающий текущий месяц. Информирует о нерестовых запретах (запрет моторов), нормах вылова (5 кг) и уголовной ответственности за использование сетей и электроудочек.
3. **Метео-модуль:** Интеграция с Open-Meteo API. Динамический запрос данных о температуре и ветре при клике на любую акваторию.
4. **Модуль личных меток:** Сохранение пользовательских гео-меток через браузерное хранилище (`LocalStorage API`) с защитой от потери данных при закрытии вкладки.

## 🛠 Технологический стек
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS, ES6+).
* **Картография:** MapLibre GL JS (мощный движок рендеринга на базе WebGL).
* **Тайлы:** Спутниковая подложка ArcGIS Satellite.
* **Формат данных:** GeoJSON.
* **Внешние API:** Open-Meteo (погода), OSRM Routing Machine (маршруты).

## 🚀 Инструкция по развертыванию
Система является полностью клиентской (serverless) и не требует сложной настройки бэкенда для базовой эксплуатации.

1. **Локальный запуск (Разработка):**
   * Откройте директорию проекта в редакторе кода (Visual Studio Code).
   * Запустите файл `index.html` с использованием локального веб-сервера (например, расширение Live Server).
2. **Публикация (Deployment):**
   * Выгрузите проект (файлы `.html`, `.geojson`) в репозиторий GitHub.
   * Перейдите в Settings -> Pages и активируйте развертывание ветки `main`.
   * Приложение станет доступно по статической ссылке и сможет работать в качестве веб-приложения на смартфонах пользователей.
