# 🎣 Fishing Map Pro — Информационно-навигационная система by schuller99

> **ℹ️ О проекте:** Данное веб-приложение выросло из тестового MVP в полноценную геоинформационную систему (ГИС) для рыболовов. Проект решает проблему разрозненности данных, объединяя навигацию, погодные условия и актуальную правовую базу Росрыболовства в едином интерфейсе.

## 📊 Анализ аудитории и проблематика
Интерфейс и функционал системы спроектированы на основе парсинга и анализа базы данных рыболовного сообщества (выгрузка от 29.03.2026). Аналитика показала следующие метрики ядра целевой аудитории:
* **Потенциальная база:** 32 532 пользователя.
* **Гендерное распределение:** 87% — мужчины.
* **Средний возраст:** 41.3 года.
* **Основная география:** Саратов и Энгельс (русло р. Волга).

Учитывая возрастную группу и условия использования (на воде, яркое солнце, нестабильная связь), интерфейс сделан максимально крупным, контрастным и интуитивно понятным, без лишних перегружающих элементов.

## 🌟 Ключевой функционал

* **Интеграция с навигаторами:** Прямое построение маршрутов до точек лова через приложения **Яндекс.Карты** и **2ГИС**, а также встроенный внутренний маршрутизатор (OSRM API).
* **Правовой модуль (Рыбнадзор):** Динамический статус-бар, который автоматически меняет предупреждения в зависимости от текущего месяца (нерестовые запреты, ограничения на использование лодочных моторов, нормы вылова в 5 кг, запрет электроудочек и сетей).
* **Метео-аналитика и Индекс клева:** При клике на водоем приложение делает запрос к *Open-Meteo API*, выводя температуру, скорость ветра и автоматически высчитывая шанс успешного клева.
* **База данных мест лова:** Использование проверенных точек, экспортированных из QGIS (`spots.geojson`), с фильтрацией по видам рыб (например, Судак).
* **Личный блокнот (Local Storage API):** Возможность сохранять собственные гео-метки ям прямо с телефона. Точки хранятся в памяти устройства и остаются доступны после перезагрузки.
* **Геолокация:** Точное позиционирование пользователя на воде через GPS устройства.

## 🛠 Технологический стек

* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Картография:** [MapLibre GL JS](https://maplibre.org/) (высокопроизводительный движок на WebGL) + ArcGIS Satellite (спутниковые снимки).
* **Маршрутизация:** OSRM (Open Source Routing Machine) + Deep links для мобильных навигаторов.
* **Данные:** GeoJSON, LocalStorage API.

## 📂 Структура проекта

* `index.html` — ядро проекта. Содержит структуру страницы, стили оформления (UI), логику интерактивных панелей, работу с API и всплывающее окно документации.
* `spots.geojson` — статичная база данных рыболовных мест.
* `README.md` — текущий файл документации.

## 🚀 Запуск и хостинг

### 1. Локальный запуск (Разработка)
1. Откройте папку с проектом в **Visual Studio Code**.
2. Запустите `index.html` через расширение **Live Server**.

### 2. Хостинг (GitHub Pages)
Проект полностью оптимизирован для работы в сети без использования сторонних backend-серверов. Все вычисления производятся на стороне клиента. 
Для обновления проекта достаточно сделать `git commit` и `git push` в ветку `main`.
