# 🎣 Интерактивная Карта Клева (Fishing Map Web) by Schuller

> **⚠️ Примечание:** Данная карта является тестовой версией (MVP). Проект находится в стадии активной разработки и будет дорабатываться и улучшаться в дальнейшем.

Это веб-приложение для рыболовов, объединяющее мощь профессиональных ГИС-систем (**QGIS**) и современные веб-технологии (**MapLibre GL JS**). Карта позволяет визуализировать рыбные места, следить за погодой и сохранять собственные точки в реальном времени.

## 🌟 Основные возможности

* **Интеграция с QGIS:** Прямое использование данных, экспортированных в формате GeoJSON.
* **Умная визуализация:** Автоматическая раскраска точек в зависимости от вида рыбы (Судак — синий, Щука — зеленый, остальные — красный).
* **Реальная погода (API):** При клике на любую точку водоема приложение отправляет запрос к сервису *Open-Meteo* и выводит текущую температуру и скорость ветра.
* **Личный блокнот (Local Storage):** Возможность добавлять свои «секретные» ямы прямо с телефона. Точки сохраняются в памяти браузера и не пропадают после перезагрузки.
* **Геолокация:** Кнопка поиска себя на карте для точного позиционирования на воде.
* **Ограничение области:** Карта настроена на Саратовскую область (р. Волга) с жесткими границами, чтобы не терять фокус.

## 📂 Структура проекта

* `index.html` — «сердце» проекта. Содержит структуру страницы, стили оформления и JavaScript-логику работы с картой и API.
* `spots.geojson` — файл базы данных, экспортированный из QGIS. Содержит координаты и характеристики (название, вид рыбы, глубина) основных точек.
* `README.md` — описание проекта и инструкции.

## 🛠 Технологический стек

* **Frontend:** HTML5, CSS3, JavaScript (ES6+).
* **Картография:** [MapLibre GL JS](https://maplibre.org/) (высокопроизводительный движок на WebGL).
* **Данные:** GeoJSON, LocalStorage API.
* **Внешние сервисы:** Open-Meteo API (погода), ArcGIS Satellite (спутниковые снимки).

## 🚀 Как использовать

### 1. Локальный запуск
1. Склонируйте или скачайте папку с проектом.
2. Откройте `index.html` через расширение **Live Server** в VS Code (или просто перетащите файл в браузер).

### 2. Обновление точек из QGIS
Если вы добавили новые точки в QGIS:
1. Нажмите правой кнопкой на слой -> Экспорт -> Сохранить объекты как...
2. Выберите формат **GeoJSON**.
3. Сохраните файл под именем `spots.geojson` в папку проекта, заменив старый файл.

### 3. Хостинг (GitHub Pages)
Проект оптимизирован для работы на GitHub Pages. Рабочая версия карты доступна по ссылке: `https://[твой-ник].github.io/[название-репозитория]/` (не забудь вставить сюда свою реальную ссылку после публикации).
